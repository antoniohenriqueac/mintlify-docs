# Configure em menos de 10 minutos ⚡

Seja bem-vindo! Este guia irá te ajudar a fazer sua primeira chamada à API ConnectVets Notes e processar um áudio de consulta veterinária.

<Info>
**Pré-requisitos**: Você precisará de uma conta ConnectVets Notes. [Crie sua conta gratuita](https://notes.connectvets.com.br/signup) se ainda não possui uma.
</Info>

## Passo 1: Obtenha suas credenciais

<Steps>
  <Step title="Faça login no dashboard">
    Acesse [notes.connectvets.com.br](https://notes.connectvets.com.br) e faça login
  </Step>
  <Step title="Navegue até API Keys">
    No menu lateral, clique em **"Configurações"** → **"API Keys"**
  </Step>
  <Step title="Crie uma nova API Key">
    Clique em **"Nova API Key"**, defina um nome descritivo e copie a chave gerada
  </Step>
</Steps>

<Warning>
**Importante**: Guarde sua API Key em segurança. Ela não será exibida novamente após a criação.
</Warning>

## Passo 2: Teste a conexão

Vamos verificar se tudo está funcionando com uma chamada simples:

<CodeGroup>
```bash cURL
curl -X GET "https://api.connectvets.com.br/v1/notes" \
  -H "X-API-KEY: your_api_key_here" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.connectvets.com.br/v1/notes', {
  method: 'GET',
  headers: {
    'X-API-KEY': 'your_api_key_here',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

headers = {
    'X-API-KEY': 'your_api_key_here',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://api.connectvets.com.br/v1/notes',
    headers=headers
)

print(response.json())
```

```go Go
package main

import (
    "fmt"
```

## Passo 3: Envie seu primeiro áudio

Agora vamos processar um arquivo de áudio. Você pode usar um áudio de teste ou uma gravação real:

<CodeGroup>
```bash cURL
curl -X POST "https://api.connectvets.com.br/v1/notes" \
  -H "X-API-KEY: your_api_key_here" \
  -F "audio=@path/to/your/audio.mp3" \
  -F "metadata={\"patient_name\":\"Rex\",\"owner_name\":\"João Silva\"}"
```

```javascript JavaScript
const formData = new FormData();
formData.append('audio', audioFile); // audioFile é um objeto File
formData.append('metadata', JSON.stringify({
  patient_name: "Rex",
  owner_name: "João Silva"
}));

const response = await fetch('https://api.connectvets.com.br/v1/notes', {
  method: 'POST',
  headers: {
    'X-API-KEY': 'your_api_key_here'
  },
  body: formData
});

const result = await response.json();
console.log('Note criada:', result);
```

```python Python
import requests

files = {
    'audio': ('audio.mp3', open('path/to/audio.mp3', 'rb'), 'audio/mpeg'),
    'metadata': (None, '{"patient_name":"Rex","owner_name":"João Silva"}')
}

headers = {
    'X-API-KEY': 'your_api_key_here'
}

response = requests.post(
    'https://api.connectvets.com.br/v1/notes',
    headers=headers,
    files=files
)

print(response.json())
```

```go Go
package main

import (
    "bytes"
    "mime/multipart"
    "net/http"
    "os"
)

func main() {
    var b bytes.Buffer
    w := multipart.NewWriter(&b)
    
    // Adicionar arquivo de áudio
    f, _ := os.Open("path/to/audio.mp3")
    defer f.Close()
    fw, _ := w.CreateFormFile("audio", "audio.mp3")
    io.Copy(fw, f)
    
    // Adicionar metadata
    w.WriteField("metadata", `{"patient_name":"Rex","owner_name":"João Silva"}`)
    w.Close()
    
    req, _ := http.NewRequest("POST", "https://api.connectvets.com.br/v1/notes", &b)
    req.Header.Set("Content-Type", w.FormDataContentType())
    req.Header.Set("X-API-KEY", "your_api_key_here")
    
    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()
}
```
</CodeGroup>

## Passo 4: Monitore o processamento

O processamento do áudio é assíncrono. Use o ID retornado para verificar o status:

<CodeGroup>
```bash cURL
# Substitua {note_id} pelo ID retornado no passo anterior
curl -X GET "https://api.connectvets.com.br/v1/notes/{note_id}" \
  -H "X-API-KEY: your_api_key_here"
```

```javascript JavaScript
async function checkStatus(noteId) {
  const response = await fetch(`https://api.connectvets.com.br/v1/notes/${noteId}`, {
    headers: {
      'X-API-KEY': 'your_api_key_here'
    }
  });
  
  const note = await response.json();
  console.log('Status:', note.data.status);
  
  if (note.data.status === 'completed') {
    console.log('Transcrição:', note.data.transcription);
    console.log('Análise:', note.data.analysis);
  }
  
  return note.data;
}
```

```python Python
import time

def wait_for_completion(note_id):
    headers = {'X-API-KEY': 'your_api_key_here'}
    
    while True:
        response = requests.get(
            f'https://api.connectvets.com.br/v1/notes/{note_id}',
            headers=headers
        )
        
        note = response.json()['data']
        print(f"Status: {note['status']}")
        
        if note['status'] == 'completed':
            print("Transcrição completa!")
            return note
        elif note['status'] == 'failed':
            print("Erro no processamento")
            return note
            
        time.sleep(5)  # Aguarda 5 segundos antes de verificar novamente
```
</CodeGroup>
