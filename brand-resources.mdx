## Diretrizes de Uso

### ✅ Permitido

- Usar em integrações oficiais com a API ConnectVets Notes
- Incluir em documentação de sistemas veterinários que usam nossa API
- Usar em interfaces que se conectam com nossa plataforma
- Redimensionar mantendo as proporções originais

### ❌ Não Permitido

- Modificar as cores ou elementos da logo
- Usar em contextos que não sejam relacionados ao ConnectVets Notes
- Distorcer ou alterar as proporções
- Usar como parte de outras marcas ou logos

## Implementação em Código

### HTML

```html
<!-- Logo responsiva -->
<img 
  src="/img/Logo_Black.png" 
  alt="ConnectVets Notes"
  style="max-width: 400px; height: auto;"
>

<!-- Logo com tema automático -->
<img 
  src="/img/Logo_Black.png" 
  alt="ConnectVets Notes"
  class="block dark:hidden"
>
<img 
  src="/img/Logo_White.png" 
  alt="ConnectVets Notes"
  class="hidden dark:block"
>
```

### CSS

```css
/* Logo responsiva */
.connectvets-logo {
  max-width: 400px;  /* Aumentado de 200px para 400px */
  height: auto;
  display: block;
}

/* Tema escuro automático */
@media (prefers-color-scheme: dark) {
  .logo-light { display: none; }
  .logo-dark { display: block; }
}

@media (prefers-color-scheme: light) {
  .logo-light { display: block; }
  .logo-dark { display: none; }
}
```

### React/Next.js

```jsx
import Image from 'next/image'

function ConnectVetsLogo({ theme = 'auto' }) {
  return (
    <div className="connectvets-logo">
      <Image
        src={theme === 'dark' ? '/img/Logo_White.png' : '/img/Logo_Black.png'}
        alt="ConnectVets Notes"
        width={400}  // Aumentado de 200 para 400
        height={120} // Proporcionalmente ajustado
        priority
      />
    </div>
  )
}
```
