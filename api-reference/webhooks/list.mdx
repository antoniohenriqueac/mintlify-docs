title: "GET /webhooks"
api: "GET https://api.connectvets.com.br/webhooks"
description: "Obter lista de webhooks configurados"
---

Listar todos os webhooks configurados na sua conta.

<RequestExample>

```bash cURL
curl -X GET https://api.connectvets.com.br/webhooks \
  -H "X-API-KEY: cvn_live_abc123def456..."
```

```javascript
const response = await fetch('https://api.connectvets.com.br/webhooks', {
  headers: {
    'X-API-KEY': 'cvn_live_abc123def456...'
  }
});

const webhooks = await response.json();
```

```python
import requests

response = requests.get(
    'https://api.connectvets.com.br/webhooks',
    headers={
        'X-API-KEY': 'cvn_live_abc123def456...'
    }
)

webhooks = response.json()
```

</RequestExample>
