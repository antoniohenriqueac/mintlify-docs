---

Criar um webhook para receber notificações automáticas sobre eventos da API.

<RequestExample>

```bash cURL
curl -X POST https://api.connectvets.com.br/webhooks \
  -H "X-API-KEY: cvn_live_abc123def456..." \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://api.meuapp.com/webhooks/connectvets",
    "events": ["note.transcription.completed"],
    "secret": "meu_segredo_webhook"
  }'
```

```javascript
const response = await fetch('https://api.connectvets.com.br/webhooks', {
  method: 'POST',
  headers: {
    'X-API-KEY': 'cvn_live_abc123def456...',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    url: 'https://api.meuapp.com/webhooks/connectvets',
    events: ['note.transcription.completed'],
    secret: 'meu_segredo_webhook'
  })
});

const webhook = await response.json();
```

```python
import requests

response = requests.post(
    'https://api.connectvets.com.br/webhooks',
    headers={
        'X-API-KEY': 'cvn_live_abc123def456...',
        'Content-Type': 'application/json'
    },
    json={
        'url': 'https://api.meuapp.com/webhooks/connectvets',
        'events': ['note.transcription.completed'],
        'secret': 'meu_segredo_webhook'
    }
)

webhook = response.json()
```
