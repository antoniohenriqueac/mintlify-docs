Listar todas as chaves de API da sua conta.

<RequestExample>

```bash cURL
curl -X GET https://api.connectvets.com.br/api-keys \
  -H "X-API-KEY: cvn_live_abc123def456..."
```

```javascript
const response = await fetch('https://api.connectvets.com.br/api-keys', {
  headers: {
    'X-API-KEY': 'cvn_live_abc123def456...'
  }
});

const apiKeys = await response.json();
```

```python
import requests

response = requests.get(
    'https://api.connectvets.com.br/api-keys',
    headers={
        'X-API-KEY': 'cvn_live_abc123def456...'
    }
)

api_keys = response.json()
```
