---

Criar uma nova chave de API para autenticação nas requisições.

<RequestExample>

```bash cURL
curl -X POST https://api.connectvets.com.br/api-keys \
  -H "X-API-KEY: cvn_live_abc123def456..." \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Integração DoctorVet",
    "type": "write",
    "rate_limit_per_minute": 100,
    "rate_limit_per_hour": 1000
  }'
```

```javascript
const response = await fetch('https://api.connectvets.com.br/api-keys', {
  method: 'POST',
  headers: {
    'X-API-KEY': 'cvn_live_abc123def456...',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    name: 'Integração DoctorVet',
    type: 'write',
    rate_limit_per_minute: 100,
    rate_limit_per_hour: 1000
  })
});

const apiKey = await response.json();
```

```python
import requests

response = requests.post(
    'https://api.connectvets.com.br/api-keys',
    headers={
        'X-API-KEY': 'cvn_live_abc123def456...',
        'Content-Type': 'application/json'
    },
    json={
        'name': 'Integração DoctorVet',
        'type': 'write',
        'rate_limit_per_minute': 100,
        'rate_limit_per_hour': 1000
    }
)

api_key = response.json()
```
