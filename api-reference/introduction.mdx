## Visão Geral

A API ConnectVets Notes oferece endpoints para transcrição de áudio e análise inteligente de consultas veterinárias. Esta seção contém a documentação completa de todos os endpoints disponíveis.

## Estrutura da API

A API é organizada em categorias principais:

### Notes
Endpoints para criar, listar e obter notas de consultas veterinárias.

### API Keys
Gerenciamento de chaves de API para autenticação.

### Webhooks
Configuração de webhooks para receber notificações de eventos.
